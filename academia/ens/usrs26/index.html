USRS26: Microcontrôleurs: architecture et communication

Informations pratiques

  • Horaires: Lundi 9:00-12:20 et jeudi 13:30-16:50
  • Emplacement: Salle 31.1.01
  • Installation de la toolchain sur vos machines:
    • MacOS:
      • Installer le gestionnaire de package Homebrew
      • brew tap PX4/homebrew-px4 && brew update && brew install gcc-arm-none-eabi stlink
    • Windows: (merci à Alan Vidal)
      • Installer le sous-système Debian
      • Installer unzip, make, cmake, git, libusb-1.0-0-dev
      • Monter la partition correspondant a la carte : sudo mount -t drvfs D: /mnt/d
      • Compiler stlink, installer les binaires et ajouter les règles udev: instructions ici
